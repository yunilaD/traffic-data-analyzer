## 🚦 Traffic Data Analyzer

A Python-based application developed to analyze traffic flow data from multiple road junctions. The system processes CSV datasets, generates key insights, and visualizes traffic patterns using histograms.

### 🔧 Features
- Input validation for date-based dataset selection
- Traffic data analysis (vehicles, trucks, electric vehicles, etc.)
- Detection of peak traffic hours
- Speed limit violation tracking
- Automated report generation (text file)
- Histogram visualization using Tkinter
- Loop system to analyze multiple datasets

### 📊 Technologies Used
- Python
- CSV File Handling
- Tkinter (for GUI & histogram visualization)

### 📁 Use Case
Designed for analyzing real-world traffic datasets to support decision-making in traffic management systems.
